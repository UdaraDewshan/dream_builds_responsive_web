## 📌 Project Overview
**Dream Builds** is a high-end architectural and luxury interior design landing page, developed as **Project 01** of the Full-Stack Internship at DecodeLabs. This project focuses on building a robust, pixel-perfect, and fully responsive "frontend interface" without the use of any modern CSS/JS frameworks.

It strictly adheres to the **"2025 Aesthetics"** guidelines, emphasizing warmth, grounding, and cinematic visual design.

## 🚀 Key Features

* **Zero Frameworks:** Built entirely from scratch using Pure HTML5, CSS3, and Vanilla JavaScript. No Bootstrap, Tailwind, or React used.
* **Mobile-First Responsive Design:** Fluid layout that perfectly adapts from mobile screens to large desktop monitors using Flexbox and CSS Grid.
* **Semantic HTML5:** Structured with accessibility and SEO in mind, utilizing proper landmark tags (`<header>`, `<main>`, `<article>`, `<footer>`).
* **2025 Aesthetics Implementation:**
  * **Color Palette:** Strictly adheres to the provided guidelines using Mocha Mousse (`#A8856F`), Ethereal Blue (`#A0D4E0`), and Moonlit Grey (`#F2F0EA`).
  * **Typography:** Clean and modern typography pairing using *Montserrat* (Headlines) and *Roboto* (Body text).
* **Interactive UI/UX:**
  * Custom Mobile Hamburger Menu with smooth toggle animation.
  * Sticky Navigation Bar with active state tracking on scroll.
  * Custom dynamic JavaScript Modal (Popup) for project details.
